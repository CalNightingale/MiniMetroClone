Small Subway

Based with love off of Mini Metro

Help the travelers reach their destination! Click and drag from a station to create a subway line.

Project Info and Background

I started this project mainly to learn more about web app development and Typescript. I really enjoyed the namesake game, Mini Metro, and decided to try to remake it to gain some valuable experience. I'm pretty happy with how this turned out, although there are some definite areas I would like to improve. Check out the issues page on the game's Github repo if you're curious about the next steps!
